# react hooks ssr 工程化的项目
- node v10.15
- 开发环境，只启动前端项目，通过webpack devserver启动，
- 生产环境，将 node 与 前端项目一起打包，通过 pm2 启动

## 运行项目

```
npm install
npm start
```

## 目录结构

```
.
├── cli  脚手架
│   └── ...
├── node_modules
├── scripts 脚本
│   └── ...
├── dist 打包后的文件
│   ├── assets/
│   │   └── ...  .js|.css
│   └── server.js
├── src
│   ├── client 客户端

│   │   │   └── common  公共组件
│   │   ├── static  静态资源
│   │   ├── app.js 前端入口
│   │   ├── index.html 前端模板
│   │   └── ...
│   ├── lib  客户端服务端公共方法
│   │   └── ...

│   │   ├── middleware 中间件
│   │   │   └── index.js 在index中引入全部中间件

```
